Grab&Go

Grab & Go is a sleek food ordering web application inspired by Swiggy, built using React, Context API, Tailwind CSS, and Parcel. It offers users a smooth browsing experience of restaurant menus, high-rated filters, and a dynamic cart system — all managed seamlessly through client-side state. Designed with performance and modularity in mind, Grab & Go demonstrates real-world integration of third-party APIs and modern frontend architecture.

Sky-Cast

Sky-Cast is a real-time weather forecasting web application built using HTML, CSS, and JavaScript, powered by WeatherAPI for dynamic weather data and OpenWeather for precise geocoding. It offers users seamless location-based weather search with intelligent input suggestions and displays clearly timestamped updates for current conditions. Designed for clarity and responsiveness, Sky-Cast ensures a fast, informative, and accessible experience across devices — making it a polished showcase of both API integration and frontend development skills.

  • JavaScript
  • RESTful API

SU30 Weather

SU30 Weather App is a dynamic weather application built with React, Redux, Parcel, and CSS. It delivers real-time weather updates, 3-day forecasts with hourly breakdowns, and comprehensive AQI insights. Featuring a built-in AI assistant powered by the Gemini API, SU30 offers interactive, location-based weather guidance through natural conversation.
